# Student Performance Analyzer

A Shiny web application for analyzing student performance using interactive visualizations and statistical insights.

## 🔗 Live Application
https://csbblr.shinyapps.io/Student_Performance_Analyzer/

## 📊 Features
- Analyze student performance
- Track attendance records
- Visualize data using charts and graphs
- Interactive dashboard

## 🛠️ Technologies Used
- R Programming
- Shiny
- Data Visualization
- CSV Data Handling

## 📁 Project Structure
- app.R → Main application file
- CSV files → Dataset (students, attendance, timetable, users)

## 🎯 Objective
To provide an interactive system for analyzing student performance efficiently.
